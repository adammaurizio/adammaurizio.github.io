Langkah Pembuatan Github Pages dengan GIT

  1. Login ke GitHub dengan akun GitHub
  2. Membuat sebuh repositori baru
  3. Buka aplikasi Git
  4. Melakukan remote terhadap repositori yang ada di Github
  5. Melakukan commit terhadap file HTML, CSS, dan lain sebagainya
  6. Melakukan push file yang sudah di-commit ke dalam repositori GitHub
  7. Cek status dengan menggunakan Git
  8. Cek repositori Github untuk mengecek
  9. Setting bagian GithubPages dan kemudian link sudah bisa diakses
